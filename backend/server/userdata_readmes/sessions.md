# Sessions

This folder holds **per-session storage** for agent conversations —
uploads attached to a chat, files generated by the agent (HTML reports,
charts, edited images), and other session-scoped artifacts.

Each session lives under its own subfolder named by the session's storage
ID (a UUID), with `uploads/` and `generated/` inside.

These files are referenced from the chat history. Editing or deleting
them by hand will break the corresponding messages — manage sessions
through the app instead.
