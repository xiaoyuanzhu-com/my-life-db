# Xiaohongshu (小红书 / RED)

No official export. Getting your own content out requires scraping — but since it's your own profile, this is straightforward with an agent.

## What you can export

- Your notes (posts) with images and captions
- Liked and saved/collected notes
- Followers and following lists
- Profile metadata

## Recommended

1. No official export exists.
2. Use an agent with Playwright or the mobile API, authenticated as you.
3. Scroll your profile, save each note as JSON + media.
4. Repeat for liked and saved tabs.
